\## Incident Summary



Security Onion detected reconnaissance activity originating from a Kali Linux system targeting a Metasploitable server within a controlled lab environment.



The alert indicated that the attacker performed an Nmap scripting scan against the target system, triggering a signature-based detection from the IDS.



This activity represents the reconnaissance phase of a potential cyber attack, where an attacker attempts to identify open services and system vulnerabilities.

\## Analysis



The alert indicates that the attacker system initiated a network scan against the target system.



The Nmap scripting engine was used to perform service enumeration and gather information about the target host.



Port 8180 was identified as an active service and was probed during the scan.



This behavior is consistent with reconnaissance activity commonly observed during the initial phase of cyber attacks.



The detection was triggered by a known signature within the Suricata intrusion detection engine.



No exploitation or unauthorized access was observed during the investigation.
